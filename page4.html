How Are Fractals Made?

Different fractals are made in different ways, but the basics involves repeating an equation and feeding it back in on itself over time to create the fractal pattern. A good example of this is the process used to create the fractal produced by the chaos game. The chaos game can be used to produce a fractal called the Sierpinski triangle.

Chaos Theory and fractals

In normal language the word ‘chaos’ is used to describe confusion. Whereas In mathematical language, a chaotic system behaves in a way that is hard to predict over time. This unpredictability is produced by very slight changes to the starting conditions. A good example of a chaotic system is the weather. Making small changes in wind strength, temperature and air pressure, can create extremely different weather patterns. This phenomenon is known as the Butterfly Effect.

How are fractal image or object created?

A fractal image or object can be created by building up many small shapes. The overall fractal will take on the shape of the smaller shapes it is built out of. A good example of this is how a twig on a tree has a very similar branching pattern to the overall shape of the tree. Chaotic systems can be analysed with fractals. Chaos Theory deals with changes within a system. Fractals are used to show those changes in the form of geometrical images.
